1. Service Description

Derax provides NFC-based app locking and unlocking services, designed to help users build a "digital detox ritual" through physical interaction, improving focus and mental well-being.

  • Android: The app uses AccessibilityService to monitor foreground app state and enforce locking logic.
  • iOS: The app uses the Family Controls (Screen Time) API to enforce system-level app restrictions. This permission is used solely to execute lock commands — we do not collect, store, or upload any app usage duration, behavioral data, or personal information through it.
  • NFC: NFC tags serve as physical keys to trigger lock or unlock actions, reinforcing the user's behavioral commitment.
